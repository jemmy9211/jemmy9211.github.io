## 🚀 Local Development

### Prerequisites

- Ruby (>= 2.7.0)
- Bundler
- Git

### Setup

1. **Clone repository**
   ```bash
   git clone https://github.com/jemmy9211/jemmy9211.github.io.git
   cd jemmy9211.github.io
   ```

2. **Install dependencies**
   ```bash
   bundle install
   ```

3. **Run locally**
   ```bash
   bundle exec jekyll serve
   ```

4. **Access website**
   Open browser and visit `http://localhost:4000`
